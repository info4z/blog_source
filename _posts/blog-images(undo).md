# 三 : picgo

* 安装 picgo

  ```shell
  $ cnpm install picgo -g
  ```

* 查看版本号

  ```shell
  $ picgo -v
  1.5.0
  ```

* 安装插件

  ```shell
  # gitee插件: picgo install gitee-uploader
  $ picgo install github-plus
  ```
